MySQL 5.1.63 がリリースされました。<br>
<br>
<br>
MySQL 5.1シリーズは、ひとつ前のメジャーバージョンですので、<br>
これからMySQLを導入される方は、最新メジャーバージョンである、<br>
MySQL 5.5 シリーズの使用をお勧めします。<br>
<br>
<br>
